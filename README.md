# Pokédex Python 🐲

Sistema de Pokédex interactivo. Día 6 de mi ruta aprendiendo Python.

### Funciones principales
1. **Ver Pokédex** - Lista todos los Pokémon con nombre, tipo, HP y ataque
2. **Capturar** - Agrega Pokémon nuevos a la base de datos
3. **Buscar** - Encuentra Pokémon por nombre sin importar mayúsculas
4. **Salir** - Guarda y cierra la Pokédex

### Conceptos de Python aplicados
- Diccionarios `{clave: valor}` para datos estructurados
- Listas de diccionarios como base de datos
- Acceso a valores con `diccionario['clave']`
- `int()` para convertir input a número
- `.lower()` para búsquedas case-insensitive
- `for` + `enumerate()` para recorrer y numerar
- Banderas booleanas `True/False`
- `f-strings` para mostrar variables
